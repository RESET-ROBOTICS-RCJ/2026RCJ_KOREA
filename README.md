RESET Robotics RCJ 2026 Korea
Official repository for Team RESET's RoboCupJunior Soccer robot developed for the RoboCupJunior Korea Open 2026.

Repository Structure
2026RCJ_KOREA/
├── HARDWARE/     # Mechanical design, CAD, PCB, manufacturing files
├── SOFTWARE/     # Robot firmware and vision software
└── README.md


---

Hardware
The HARDWARE directory contains all mechanical and electrical design files.

Contents may include:

Fusion 360 CAD models (.f3d, .f3z)
STEP files
STL files for 3D printing
PCB designs
Mechanical drawings

---

Software
The SOFTWARE directory contains all source code required to operate the robot.

Typical components include:

Main controller firmware
Vision system
Sensor drivers
Motor control
Communication modules
Utility scripts

---

Development
Hardware
CAD is designed using Autodesk Fusion 360.

Software
Languages used include:

C++
Python
Arduino
